## Результаты каскадного тестирования

### Дашборд (/)
- **Тип:** KPI-дашборд
- **Виджеты:** Сводка (лиды/сделки/товары/задачи), Пульс, Быстрые действия
- **Быстрые действия:** Новый клиент, Новая сделка, Новая задача, Написать в чат
- **Аномалии:** кнопка "Настроить" не имеет видимого действия
- **Статус:** ✅

### Чат (/chat)
- **Тип:** Мессенджер (список чатов + область сообщений)
- **Empty State:** "Нет чатов" + "Select a conversation to start chatting" ✅
- **Загрузка:** skeleton-анимация при загрузке ✅
- **Ошибка:** состояние "Ошибка загрузки" + кнопка "Повторить" ✅
- **Поиск:** фильтрация по имени ✅
- **Файлы:** загрузка через вложение ✅
- **Статус:** ✅

### Клиенты (/clients)
- **Тип:** Таблица-сетка с фильтрами
- **Фильтры:** поиск, источник, менеджер, сортировка
- **Сортировка:** название, телефон, email, ИНН, статус, источник, сделки, ответственный, дата
- **Кнопка "Сбросить фильтры":** появляется при активных фильтрах ✅
- **URL-параметры:** фильтры сохраняются в URL (?status=Active) ✅
- **Создание:** модальная форма с 8 полями
- **Детальная карточка:** сделки, задачи, быстрые действия (создать сделку/задачу)
- **Редактирование/Удаление:** ✏️🗑 в едином стиле ✅
- **Статус:** ✅

### Сделки (/deals)
- **Тип:** Канбан (8 колонок: Лид→Счёт→Юристы→Доки→Оплата→Резерв→Отгрузка→Закрыто)
- **Воронка:** двусторонняя (← предыдущий / следующий →)
- **Валидация:** clientId обязателен, красная подсветка ✅
- **Детальная карточка:** сумма, тип, клиент, агент, дата, связанные данные
- **Редактирование/Удаление:** ✏️🗑 + двухшаговое подтверждение удаления ✅
- **Фильтры:** агент, клиент
- **Статус:** ✅

### Склад (/warehouse)
- **Тип:** Дерево каталогов + таблица товаров
- **Вкладки:** Товары / Поступления / Перемещения
- **Каталоги:** создание, ✏️ редактирование, 🗑 удаление с названием в диалоге ✅
- **Товары:** создание, детальная модалка (артикул, количество, цены, маржа)
- **Индикация:** количество < 5 — жёлтый, 0 — красный ✅
- **Поступления:** заглушка "будут доступны после интеграции с 1С"
- **Статус:** ⚠️ вкладка Поступления — placeholder без функционала

### Задачи (/tasks)
- **Тип:** 3 режима (Канбан / Список / Календарь)
- **Канбан:** 4 колонки (Новая/В работе/Выполнена/Отменена)
- **Воронка:** двусторонняя (← / →) ✅
- **Выделение:** режим "Выбрать" → чекбоксы → массовое удаление ✅
- **Детальная модалка:** сквозные ссылки (сделка, клиент, исполнитель) ✅
- **Календарь:** месяц + список ближайших, клик по дню → создание
- **Статус:** ✅

## Баги и аномалии

### Средние (⚠️)
1. **Склад — Поступления:** вкладка без функционала, placeholder "после интеграции с 1С"
2. **Сервис:** раздел показывает контент но данных нет (пустой)

### Косметические (💅)
1. **Дашборд — кнопка "Настроить":** не реагирует на клик
2. **Производство / Монтажи / Договоры:** пустые состояния без CTA-кнопок создания
3. **Скрытые разделы:** /products и /knowledge не в боковом меню (доступны только по URL)
